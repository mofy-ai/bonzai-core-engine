Class ClaudeCliManager

Advanced Claude CLI session manager with intelligent error handling

This class provides comprehensive management of Claude CLI sessions with advanced features:

  • Intelligent retry strategies with exponential backoff and jitter
  • Circuit breaker patterns for preventing cascade failures
  • Memory-optimized output buffering with size limits
  • Real-time performance monitoring and health checks
  • Advanced error recovery with graceful degradation
  • Resource limit enforcement and automatic cleanup
  • Telemetry collection and session analytics

ClaudeCliManager

Example

const manager = new ClaudeCliManager(configManager);
const result = await manager.executeClaudeCommand(
  "Analyze this codebase and suggest improvements",
  {
    files: ["src/main.ts"],
    timeout: 120000,
    enableRecovery: true,
    maxRetries: 3
  }
);
