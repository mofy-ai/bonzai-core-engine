Class TrueNorthOrchestrator

Main orchestrator for TrueNorth intelligent agent system

This class serves as the central coordinator for all TrueNorth operations, managing the complete workflow from task selection through execution. It handles:

  • Task lifecycle management
  • Agent system coordination
  • Phase-based execution orchestration
  • Real-time dashboard integration
  • Error recovery and system health monitoring

TrueNorthOrchestrator

Example

const orchestrator = new TrueNorthOrchestrator(claudeCliManager, configManager);
orchestrator.setDashboardManager(dashboardManager);
await orchestrator.executeTask('clean_codebase');
