ğŸ”§ Troubleshooting

Extension not working?

  1. Ensure Claude CLI is installed: claude --version
  2. Check VSCode version: code --version (needs 1.74.0+)
  3. Open VSCode Developer Console: Help > Toggle Developer Tools

Agents failing?

  1. Check Claude CLI configuration and API keys
  2. Ensure project has proper file permissions
  3. Check the True North Dashboard for detailed error messages
  4. Use "Retry Failed Agents" command

Dashboard not loading?

  1. Try closing and reopening the dashboard
  2. Reload VSCode window: Ctrl+R (Mac: Cmd+R)
  3. Check for extension updates
